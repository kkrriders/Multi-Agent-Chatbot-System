**Create New:**
```
Click: "+ New" button
```

---

### Dark Mode:

**Light Mode:**
- White backgrounds
- Dark text
- Blue/purple accents
- Perfect for daytime

**Dark Mode:**
- Dark gray backgrounds
- Light text
- Vibrant accents
- Easy on eyes at night

**System Mode:**
- Follows OS setting
- Auto-switches at sunset (if OS does)
- Best of both worlds

## Tips & Tricks

### Conversation Management:
- **Organize:** Use descriptive titles for easier search
- **Archive:** Old conversations to keep sidebar clean
- **Search:** Find conversations quickly
- **Delete:** Remove unwanted conversations (confirm first!)

### Dark Mode:
- **Battery life:** Dark mode can save battery on OLED screens
- **Eye strain:** Use dark mode in low-light environments
- **Consistency:** System mode matches your OS theme
